Sobre o Projeto

O IFSC (Instituto Federal de Santa Catarina) oferece uma oficina prática e interativa sobre Git e GitHub, voltada para alunos e profissionais que desejam aprender a controlar versões e colaborar em projetos de desenvolvimento de software. Durante a oficina, os participantes aprenderão desde os conceitos básicos do Git, como criação de repositórios e commits, até o uso avançado do GitHub para gerenciar projetos colaborativos, contribuir com código e entender fluxos de trabalho eficientes. Ao final, os participantes terão uma compreensão sólida das ferramentas, essenciais para o desenvolvimento de software moderno.
